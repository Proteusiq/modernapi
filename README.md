
# Modern API

Building secured API with `FastAPI` and `SQLModel`




## Installation

Install mapi with `poetry`

```bash
git clone https://github.com/Proteusiq/modernapi.git && cd modernapi
poetry install
invoke init-db
cd mapi
uvicorn main:app --reload  
```
